Mapper接口和Mapper映射文件

Mapper接口

public interface EmployeeMapper {
    List<Employee> findEmp();

    List<Employee> findEmpWithDeptno(int deptno);

    List<EmpExt> findEmpAndDept(int id);

    List<EmpExt> findEmpAndDept2();
}

注：此处我们可以发现，mapper接口与Mapper映射文件中的方法名一一对应，parameterType类型以及resultType类型也是一一对应的，同时需要注意的是，我们借助了resultMap元素描述了一对多的映射关系，在一对多的查询中，一的一方必然持有一个容器包含若干多的一方，那么就会形成关系映射，需要用Map表明这种关系，EmpExt这个类则在实体类上表明了这种关系
