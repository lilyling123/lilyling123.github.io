视图与简介

关系视图

简介

Iterable

迭代器模式的接口，凡是实现了此接口的类都会提供一个迭代器，并且可以使用forEach

Iterator<T> iterator();

Iterator


提供一个迭代器，可以将集合迭代，如何实现取决于具体的类，主要方法如下：

boolean hasNext();//是否还有下一个元素
E next();//指针指向并返回下一个元素
default void remove() {
    throw new UnsupportedOperationException("remove");
}            //提供了默认的实现

ListIterator


提供一个可以改变指向的迭代器，只有List相关类实现了此接口，继承了Iterator的所有方法

boolean hasPrevious();    //是否还有上一个元素
E previous();                //返回上一个元素
int nextIndex();            //返回下一个元素下标
int previousIndex();        //返回上一个元素下标
void remove();            //移除当前元素，有可能发生异常
void set(E e);            //重新设置当前元素
void add(E e);            //在当前元素后面添加一个新元素

Collection

增删改查

boolean add(E e);                            //用于添加一个元素
boolean remove(Object o);                    //用于删除一个指定元素
boolean contains(Object o);                    //用于查询是否包含一个指定元素
boolean containsAll(Collection<?> c)    ;        //用于查询是否包含一组元素
boolean addAll(Collection<? extends E> c);    //用于添加一组元素
boolean removeAll(Collection<?> c);            //用于移除一组元素
boolean retainAll(Collection<?> c);            //用于移除c中不包含的元素（并集）

功能型

int size();                //用于查询Collection中的元素个数
boolean isEmpty();        //用于查询Collection是否为空
Iterator<E> iterator();    //提供一个Collection的迭代器

其他

Object[] toArray();            //用于将Collection转换为数组
<T> T[] toArray(T[] a);        //用于将Collection转换为指定类型的数组
void clear();                    //清空Collection
boolean equals(Object o);    //判断Collection相等
int hashCode();                //定义Collection的hashCode

Map

增删改查

V get(Object key);                                //获取键为key的值
V put(K key, V value);                            //加入一个键值对
void putAll(Map<? extends K, ? extends V> m);    //放入一个Map类型的数据
V remove(Object key);                            //删除一个指定key的元素
void clear();                                        //清空
boolean containsKey(Object key);                    //是否包含某个key
boolean containsValue(Object value);                //是否包含某个value

功能

int size();                                        //返回map规模
boolean isEmpty();                                //返回map是否为空

其他

Set<K> keySet();                                //返回map的Key的Set集合
Collection<V> values();                            //返回map的Value的Collection集合
Set<Map.Entry<K, V>> entrySet();                //返回map中的键值对set集合

说明

同时提供一系列用于不同的并且有默认实现的方法，此处暂且忽略

List


继承了Collection的所有的方法，并且有一些自己的方法，在此只介绍自己的特有的方法

增删改查

boolean addAll(int index, Collection<? extends E> c);    //用于添加一组元素
E get(int index);                                        //查询指定下标的元素
E set(int index, E element);                            //将指定下标的元素设为E，并且返回之前的元素
void add(int index, E element);                        //在指定下标添加一个元素,后面元素全部后移
E remove(int index);                                    //移除指定下标的元素
int indexOf(Object o);                                //查找指定元素的位置
int lastIndexOf(Object o);                                //从后向前查找指定元素的位置

功能

ListIterator<E> listIterator();                //提供了一个迭代器
ListIterator<E> listIterator(int index);        //根据index提供一个迭代器
List<E> subList(int fromIndex, int toIndex);//返回 List 在 fromIndex 与 toIndex 范围内的子集，注意是左闭右开

Set


与Collection接口内容完全一致，此处不再介绍

Queue

boolean add(E e);    //添加一个元素
boolean offer(E e);    //添加一个元素
E remove();            //从队首移除一个元素，并返回
E poll();                //从队首移除一个元素，并返回
E element();            //返回队首元素，不删除
E peek();            //返回队首元素，不删除
