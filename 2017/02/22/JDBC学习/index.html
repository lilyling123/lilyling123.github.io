JDBC

简介

应用程序、JDBC API、数据库驱动和数据库

java定义了一系列的操作数据库的接口，由数据库厂商提供实现这些接口的驱动

初步使用

连接数据库的步骤

  1. 注册驱动（只做一次）
  2. 建立连接
  3. 创建执行SQL的语句
  4. 执行语句
  5. 处理执行结果

  6. 释放资源

    //注册驱动
DriverManager.registerDriver(new com.mysql.jdbc.Driver());

//建立连接
Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/jdbc", "root", "123");

//创建语句
Statement st = conn.createStatement();

//执行语句
ResultSet rs = st.executeQuery("");

//处理结果:按行遍历
while (rs.next()){

}
//关闭资源
rs.close();
st.close();
conn.close();

    getConnection参数说明:URl，用户名，密码

增删改查

直接使用Statement可能会产生sql注入的问题

使用preparedStatement，在构造的时候会进行预处理

preparedStatement效率更高

数据库和驱动对PreparedStatement有优化

放置大文本到数据库中

File file = new File("D:\\BasicAlgorithm\\src\\JDBC\\JdbcUtils.java");
Reader r = new BufferedReader(new FileReader(file));

ps.setCharacterStream(1, r,(int) file.length());

ps.executeUpdate();

从数据库中读取文本数据

rs = ps.executeQuery();

while (rs.next()) {
     Clob b = rs.getClob(1);
     Reader r = b.getCharacterStream();
     BufferedReader br = new BufferedReader(r);
     String line = "";
     while ((line = br.readLine()) !=null){
         System.out.println(line);
     }
}

获取存储后的主键（主键一般是唯一的，自动生成，不可修改的）

conn.prepareStatement(sql,Statement.RETURN_GENERATED_KEYS);

rs = ps.getGeneratedKeys();

批处理

ps.addBatch();

ps.executeBatch();//返回的是一个数组

可滚动的结果集

rs.previous()

rs.absolute(150)//可以直接定位到第150条开始取数据

st = conn.createStatement(ResultSet.TYPE_SCROLL_SENSITIVE,ResultSet.CONCUR_UPDATABLE);
